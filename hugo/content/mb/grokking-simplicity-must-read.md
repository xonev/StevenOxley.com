I think [Grokking Simplicity](https://www.manning.com/books/grokking-simplicity) is a must read for anyone who works with code. Even if you're vibe coding, Grokking Simplicity can help you instruct your AI agents to generate good code rather than slop. If you actually want to apply functional concepts and understand why they're good, pick this one up.
